What is AoC Script Studio?
Write chat scripts & macros Live in-game preview 3,233 AoC icons built-in One HTML file — fully offline Runs on Windows in any browser

Script Studio is a single HTML file — no install, no account, no internet required. Open it in any browser on Windows, write your script using colours, icons and chat commands, and see exactly how it will look in-game before you paste it.

This survey has no script library built in — it opens clean and fast. One of the questions below asks whether you would want a library added. Your votes decide what gets built next.

■ Privacy: this survey does not save anything to your browser by default. Your answers exist only while the page is open. At the end you will have the option to save your response on this device — that is entirely your choice.
